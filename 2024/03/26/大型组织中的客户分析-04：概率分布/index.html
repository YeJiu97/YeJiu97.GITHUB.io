伯努利

关于伯努利试验

伯努利试验（Bernoulli trial）是一种随机实验，其结果只有两种可能：成功或失败。在统计学和概率论中，伯努利试验是最基本的随机实验之一，常用于模拟只有两个可能结果的情况。

伯努利试验遵循以下条件：

  • 试验结果只有两种，通常被标记为“成功”和“失败”；
  • 每次试验的成功概率（记为 $p$）是固定的，而失败的概率（记为 $1-p$）也随之确定；
  • 各次试验是独立的，即前一次试验的结果不影响后一次试验的结果。

伯努利试验的概率质量函数（PMF）描述了成功（$x=1$）或失败（$x=0$）的概率：

$ P(X = x) = p^x(1-p)^{1-x} $

其中，$x$ 取值为 0 或 1，$p$ 是成功的概率。

  1. 抛硬币：在抛一枚公正硬币的实验中，硬币落地时朝上的一面只能是正面或反面，这就形成了一个典型的伯努利试验。如果我们定义“硬币朝上是正面”为成功，那么成功的概率 $p = 0.5$，失败的概率自然也是 $0.5$。

  2. 抽签：从一个包含红球和蓝球的袋子中随机抽取一球，目标是抽取到红球。如果袋子中有10个球，其中3个是红球，那么成功（即抽到红球）的概率是 $p = 0.3$，而失败（即抽到蓝球）的概率是 $1-p = 0.7$。

这些例子展示了伯努利试验在不同情况下的应用，它们的共同点在于结果的二元性和每次试验独立性。

伯努利试验是一个随机试验，它只有两种可能的结果：成功或失败。在这个上下文中，”成功”不一定意味着一个好的结果，它只是试验结果的一个标签。

  • $ p $ 表示成功的概率，它在每次试验中都是相同的。
  • $ q $ 表示失败的概率，也是每次试验中都是相同的。
  • $ p $ 和 $ q $ 是互补的，这意味着 $ p + q = 1 $。这是因为成功和失败是两个互斥的完备事件，也就是说，每次试验必然是成功或失败，没有其他可能。

公式 $ p = \frac{S}{S + F} $ 描述了如何根据过去的试验结果计算成功的概率，其中 $ S $ 表示成功的次数，$ F $ 表示失败的次数。这基本上是在说，成功的概率等于成功次数除以总的试验次数。

相应地，$ q = \frac{F}{S + F} $ 描述了如何计算失败的概率。

例如，假设我们有一个硬币，我们把正面朝上称为“成功”，反面朝上称为“失败”。如果我们抛了硬币100次，得到了70次正面和30次反面，那么成功的概率 $ p $ 就是70除以100，即 $ p = 0.7 $，而失败的概率 $ q $ 就是30除以100，即 $ q = 0.3 $。由于这两个概率必须加起来等于1，所以这个结果是有意义的。

伯努利分布的概率质量函数为:

$P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}， k=0，1，2，…，n$

其中，$\binom{n}{k}$表示二项式系数，计算公式为:

$\binom{n}{k} = \frac{n!}{k!(n-k)!}$

举例:假设抛掷一枚不均匀的硬币10次，每次硬币正面朝上的概率为0.6，则抛掷10次中正面朝上的次数X服从参数n=10， p=0.6的伯努利分布。

如果我们想知道正面朝上恰好6次的概率，可以计算:

$P(X=6) = \binom{10}{6} \times 0.6^6 \times (1-0.6)^{10-6} \approx 0.2508$

因此，在这10次抛掷中，正面朝上恰好6次的概率约为25.08%。

总之，伯努利分布描述了在n次独立重复试验中，事件发生次数的概率分布规律，具有广泛的应用，如赌场中的博彩、质量控制等领域。

例子一：考试

有一个测试包含10个多项选择题，每个题目有3个可能的答案选项。假设学生对这个主题完全不了解，那么他在每个题目上选对的概率就是p=1/3。

问题是:对于这个学生来说，他的期望成绩是多少?

  1. 我们使用伯努利分布来建模这个问题。对于每个题目，学生要么选对(成功)，要么选错(失败)。
  2. 设随机变量X表示学生答对的题目数量。
  3. 由于每个题目都是相互独立的，且选对的概率都相同(1/3)，所以X服从参数n=10，p=1/3的伯努利分布。
  4. 伯努利分布的均值(期望值)等于 np，即:$\mathrm{E}[X] = n \times p = 10 \times \frac{1}{3} = \frac{10}{3} \approx 3.33$

因此，对于这个学生来说，他在这次测试中期望能够答对大约3.33个题目。

泊松分布

什么是泊松分布

泊松分布是一种统计分布，它用于描述在固定的时间间隔、空间区域、长度或体积内发生的独立事件的平均发生率（λ，lambda）已知的情况下，观察到具有特定次数的事件发生的概率。泊松分布特别适用于那些发生率较低，但是有大量观察机会的随机事件。

泊松分布的概率质量函数（Probability Mass Function PMF）为：$ P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!} $

  • $ X $ 是在给定时间间隔内发生事件的次数。
  • $ k $ 是可能观察到的事件次数，它可以是任何非负整数$0 1 2 …$。
  • $ \lambda $ 是事件在指定时间间隔内发生的平均次数（也称为速率参数）。
  • $ e $ 是自然对数的底数（约等于2.71828）。
  • $ k! $ 是 $ k $ 的阶乘。

例子一：电话

假设某个电话服务中心每小时接到2个电话，我们想要知道在下一个小时内接到3个电话的概率。如果我们假设电话的到达服从泊松过程（即电话的到达是独立的，并且有固定的平均速率），则可以使用泊松分布来计算这个概率。

在这个例子中，$ \lambda = 2 $（因为平均每小时2个电话），而我们要计算的是 $ k = 3 $（即3个电话）的概率：$ P(X = 3) = \frac{2^3 e^{-2}}{3!} = \frac{8 e^{-2}}{6} \approx 0.180 $

因此，在下一个小时内接到3个电话的概率约为18%。

负二项分布

二项分布

二项分布（Binomial Distribution）是离散概率分布的一种，描述了在固定次数的独立实验中，每次实验成功的概率都是相同的情况下，成功的次数的分布。如果我们进行了 $ n $ 次独立的伯努利试验（每次实验成功的概率是 $ p $，失败的概率是 $ 1-p $），那么成功恰好 $ k $ 次的概率可以用二项分布来描述。

二项分布的概率质量函数（Probability Mass Function, PMF）是：

$ P(X = k) = \binom{n}{k} p^k (1-p)^{n-k} $

其中：

  • $ X $ 是成功的次数。
  • $ k $ 是指定成功的次数，可以取的值是 $ 0, 1, 2, \ldots, n $。
  • $ n $ 是试验的总次数。
  • $ p $ 是每次试验成功的概率。
  • $ \binom{n}{k} $ 是组合数，表示从 $ n $ 次试验中选择 $ k $ 次成功的方式数，计算公式为 $ \binom{n}{k} = \frac{n!}{k!(n-k)!} $。

例子：如果抛掷一枚硬币10次（每次得到正面的概率为0.5），那么恰好得到5次正面的概率就是：

$ P(X = 5) = \binom{10}{5} (0.5)^5 (1-0.5)^{10-5} $

这表示从10次抛掷中选择5次正面的方式数乘以每次抛掷得到正面的概率的五次方乘以每次抛掷得到反面的概率的五次方。

另一方面，负二项分布（Negative Binomial Distribution）也是一种离散概率分布，但它描述的是一系列独立的伯努利试验直到发生指定次数的成功为止所需的失败次数。换句话说，它表示在达到一定数量的成功之前失败的次数。如果我们想知道“需要多少次失败才能获得 $ r $ 次成功”，那么就会用到负二项分布。

负二项分布的概率质量函数是：

$ P(X = k) = \binom{k+r-1}{k} p^r (1-p)^k $

这里的 $ k $ 表示失败的次数，$ r $ 是达到的成功次数，而 $ p $ 是每次实验成功的概率。请注意，此处的 $ k $ 可以是任何非负整数，包括0。这表示在获得 $ r $ 次成功之前，有 $ k $ 次失败的概率。

负二项分布

Negative binomial distribution（负二项分布）是一种离散概率分布，用于描述在一系列独立且相同分布（i.i.d.）的伯努利试验中，达到预定数量的成功所需的失败次数。换言之，它给出了在成功次数固定时，失败次数的概率分布。

负二项分布通常用两个参数来描述：

  • $ r $：表示要观察到的成功次数。
  • $ p $：每次试验中成功的概率。

当我们重复进行伯努利试验，直到发生了 $ r $ 次成功为止，负二项分布给出了在这个过程中失败次数 $ k $ 的概率分布。其概率质量函数（PMF）如下：

$ P(X = k) = \binom{k + r - 1}{r - 1} p^r (1 - p)^k $

这里 $ X $ 是在第 $ r $ 次成功之前观察到的失败次数，$ k $ 是可能观察到的失败次数，它可以是任何非负整数（0, 1, 2, …）。

负二项分布在模型预测、生态学和医学研究中非常有用，尤其适合于那些发生率不是很高或者不确定的场景。

例子：

均匀分布

均匀分布（Uniform Distribution）是一种概率分布，其中所有事件发生的概率相等。均匀分布可以是离散的，也可以是连续的。

离散均匀分布

在离散均匀分布中，每个离散结果发生的概率是相同的。如果有 $ n $ 个离散的结果，每个结果发生的概率就是 $ \frac{1}{n} $。

例如，掷一个公平的六面骰子，每个面朝上的概率都是 $ \frac{1}{6} $。

连续均匀分布

连续均匀分布涉及到一个连续的区间$a, b$，在这个区间内的任何值被选中的概率密度是相同的。其概率密度函数（Probability Density Function, PDF）为：

$ f(x) = \frac{1}{b-a} \quad \text{for} \quad a \le x \le b $

其中，$ a $ 和 $ b $ 是区间的下限和上限，且 $ a < b $。在整个区间内，概率密度是常数。

均匀分布的期望值（平均值）是 $\frac{a + b}{2}$，方差是 $\frac{(b-a)^2}{12}$。

连续均匀分布的例子：假设一个随机变量 $ X $ 描述了一辆车在某个时间段内到达的时间，如果这辆车在8点到9点之间等概率地在任何时间到达，那么 $ X $ 就服从区间 $[8, 9]$ 上的连续均匀分布。

指数分布

是什么

指数分布是一种连续概率分布，常用于描述在两个连续事件之间的时间间隔。它是泊松过程的一部分，泊松过程是一种描述在固定时间间隔或空间内发生随机事件的数目的模型。指数分布特别适用于描述无记忆的过程，这意味着未来的事件发生不依赖于过去发生的事件。

指数分布的概率密度函数（PDF）为：$ f(x;\lambda) = \lambda e^{-\lambda x} \quad \text{for } x \geq 0 $

其中：

  • $x$ 是随机变量，表示两次事件之间的时间间隔。
  • $\lambda$ 是事件发生的平均率（即单位时间内事件发生的次数），$\lambda > 0$。
  • $e$ 是自然对数的底数。

指数分布的累积分布函数（CDF）为：$ F(x;\lambda) = 1 - e^{-\lambda x} \quad \text{for } x \geq 0 $

指数分布的特点包括：

  • 无记忆性：指数分布的一个关键特性是无记忆性，这意味着未来的预测不依赖于已经过去的时间。具体来说，如果某个过程遵循指数分布，那么从任何时间点开始，到下一个事件发生的分布仍然是相同的。
  • 平均值和方差：指数分布的平均值（或期望值）是 $\frac{1}{\lambda}$，方差是 $\frac{1}{\lambda^2}$。

例子：假设一个呼叫中心接到电话的平均速率是每小时3个电话，那么我们可以用指数分布来模拟在接到一个电话后，接到下一个电话的时间间隔。在这个例子中，$\lambda = 3$，表示平均每小时发生3次事件（电话）。根据指数分布，我们可以计算在接到第一个电话后10分钟内接到另一个电话的概率等。

正态分布

什么是正态分布

正态分布，也被称为高斯分布，是连续概率分布的一种，它在自然科学和社会科学中非常普遍。正态分布的图形呈现为对称的钟形曲线，这种分布的特点是数据围绕中心值对称分布，大多数数据集中在中心附近，越往两侧分布的数据越少。

正态分布由两个参数确定：均值（μ）和标准差（σ）。均值决定了分布的中心位置，标准差决定了分布的宽度，即数据的集中或分散程度。

正态分布的概率密度函数（Probability Density Function, PDF）公式为：$ f(x|\mu,\sigma) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} $

其中：

  • $ x $ 是变量
  • $ \mu $ 是均值
  • $ \sigma $ 是标准差
  • $ \sigma^2 $ 是方差
  • $ e $ 是自然对数的底（约等于2.71828）
  • $ \pi $ 是圆周率（约等于3.14159）

正态分布的一些重要性质包括：

  • 约68%的数据在均值加减一个标准差（$ \mu \pm \sigma $）的范围内
  • 约95%的数据在均值加减两个标准差（$ \mu \pm 2\sigma $）的范围内
  • 约99.7%的数据在均值加减三个标准差（$ \mu \pm 3\sigma $）的范围内

这种分布在实际应用中非常常见，比如测量误差、人类的身高、考试成绩等都近似遵循正态分布。

转换为正态分布

将非正态分布的数据转换为正态分布是统计分析和数据科学中的一个常见步骤，原因主要包括以下几点：

  1. 统计测试的假设：许多统计测试（如t检验、ANOVA）和参数估计方法都基于正态分布的假设。这些方法假设数据或数据的误差项是正态分布的。如果数据显著偏离正态分布，这些方法的有效性和准确性可能会受到影响。

  2. 线性回归模型的假设：线性回归和许多其他回归方法假设残差（实际观测值与预测值之间的差异）是正态分布的。如果残差的分布显著偏离正态分布，可能会影响模型的解释和预测的准确性。

  3. 提高算法性能：某些机器学习算法对数据的分布有特定的要求。将数据转换为更接近正态分布可以改善算法的性能，特别是对于那些基于数据分布特性的算法，如线性模型和支持向量机。

  4. 数据变换的效果：对数据进行正态化处理可以使其更加稳定和均匀，有时还可以减少数据中的极端值（outliers）对分析结果的影响，使得数据分析结果更加可靠。

  5. 方便数据的解释：正态分布是最广为人知的概率分布之一，其特性和性质被广泛理解。将数据转换为正态分布，可以让非专业人士更容易理解数据的统计特性，比如均值和标准差等统计量。

常用的使数据正态化的方法包括对数转换（log transformation）、平方根转换（square root transformation）、Box-Cox转换等。选择哪种转换方法取决于数据的具体情况和分布的特性。

分布之间的关系

这张图展示了不同概率分布之间的关系，并指出了一些分布作为其他分布的特例或极限形式。

  • 均匀分布（Uniform）：通常是其他分布的起点，特别是当离散分布的结果数量变得非常大时，它可以近似为其他分布。
  • 伯努利分布（Bernoulli）：描述了只有两种可能结果的单次随机试验（例如，抛硬币）。
  • 二项分布（Binomial）：由一系列相同的伯努利试验组成，其结果是成功次数的分布。
  • 几何分布（Geometric）和负二项分布（Negative Binomial）：与二项分布相关，几何分布描述了第一次成功之前失败次数的分布，而负二项分布描述了达到固定数量成功之前的失败次数的分布。
  • 超几何分布（Hypergeometric）：类似于二项分布，但是试验不是独立的，因为每次试验后成功的概率会改变。
  • 泊松分布（Poisson）：描述了在给定时间或空间内事件发生次数的分布，是二项分布在试验次数很大而成功概率很小的极限情况。
  • 指数分布（Exponential）：描述了两个连续泊松事件之间的时间间隔，是几何分布的连续形式。
  • 正态分布（Normal, Gaussian）：也被称为高斯分布，是许多分布的极限分布，特别是当独立随机变量的数量增多时（根据中心极限定理）。
  • 对数正态分布（Log Normal）：当一个变量是多个独立正态分布变量的乘积的对数时，它呈对数正态分布。
  • 学生t分布（Student’s t）：当估计正态分布的均值而标准差未知时使用，尤其是样本量较小的时候。
  • 卡方分布（Chi-Squared）：是多个独立的标准正态分布变量的平方和，通常用于假设检验和置信区间的计算。
  • 韦伯分布（Weibull）：是一种可靠性分析和生存分析中常用的分布，形
